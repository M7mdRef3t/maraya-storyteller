# مرايا — نظام التصميم (v1.0.0)

> **Maraya Design System** — بناء تجربة سردية بصرية سينمائية

---

## الرؤية

مرايا ليست تطبيقًا — إنها **مرآة رقمية** تعكس المشاعر وتحوّلها إلى قصص حيّة.
نظام التصميم مبنيّ ليُشعر المستخدم أنه داخل عالم سينمائي يتنفّس مع كل تفاعل.

## المبادئ الأساسية

### 1. 🪞 الانعكاس (Reflective)
كل عنصر واجهة يعكس حالة المستخدم العاطفية. الألوان والحركة والنصوص تتغير بحسب السياق.

### 2. 🌌 الغمر (Immersive)
التصميم يذوب — لا يوجد حدود صلبة. الزجاج الضبابي (Glassmorphism) والتدرجات الشفافة تخلق عمقًا سينمائيًا.

### 3. ✨ الحياة (Alive)
كل شيء يتحرّك بنعومة: عناصر تطفو، نبضات ضوء، انتقالات سلسة. الواجهة كائن حيّ وليست صفحة ساكنة.

### 4. 🎯 الوضوح (Clarity)
رغم الغنى البصري، المحتوى يبقى الملك. التسلسل الهرمي واضح، والقراءة سهلة، والتفاعل بديهي.

### 5. ♿ الشمولية (Inclusive)
كل مكوّن يدعم قابلية الوصول: تباين WCAG AA+، دعم قارئات الشاشة، تركيز لوحة المفاتيح، RTL أصيل.

### 6. 🎬 السينمائية (Cinematic)
نحن نبني تجربة عرض سينمائي — لا مجرّد واجهة. كل لحظة في الرحلة مصمّمة كمشهد.

## التقنيات المستخدمة

- **Frontend**: React + Vite
- **Styling**: Vanilla CSS مع CSS Custom Properties (Design Tokens)
- **Fonts**: Google Fonts (Cairo, Outfit)
- **Icons**: Emoji-based (للتوافق العالمي)
- **Animation**: CSS Animations + Transitions (خفيفة، بدون مكتبات)
- **RTL**: CSS Logical Properties (`inset-inline-end`, `border-inline-start`...)
